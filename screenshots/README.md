# Screenshots

This folder contains screenshots and demo GIF for VideoGrabber.

## Files

- `demo.gif` - Demo animation showing the app in action
- `main.png` - Main interface screenshot
- `queue.png` - Download queue screenshot
- `history.png` - History management screenshot

## How to Add Real Screenshots

1. **Take Screenshots**
   - Open VideoGrabber app
   - Take screenshots of main interface, queue, and history
   - Record a 15-30 second demo GIF

2. **Optimize Images**
   - Resize to reasonable dimensions (800-1200px wide)
   - Compress to reduce file size
   - Use PNG for screenshots, GIF for demo

3. **Replace Placeholders**
   - Replace the placeholder files in this folder
   - Commit and push to GitHub

## Tools for Screenshots

- **macOS**: Cmd+Shift+4 (screenshot), Cmd+Shift+5 (screen recording)
- **Windows**: Win+Shift+S (screenshot), Win+G (screen recording)
- **GIF Creation**: [LICEcap](https://www.cockos.com/licecap/), [Kap](https://getkap.co/)

## Recommended Dimensions

- Demo GIF: 800x600px, 15-30 seconds, <5MB
- Screenshots: 1200x800px, PNG format, <500KB each
