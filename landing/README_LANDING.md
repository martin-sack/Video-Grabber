## â“ FAQ

### Is VideoGrabber really free?
Yes! 100% free and open source (MIT license). No ads, no premium tiers, no hidden costs.

### Does it work offline?
Yes! All dependencies are bundled. Once installed, you only need internet to download videos.

### Is it safe?
Yes! Open source code you can inspect. No data collection, no telemetry, no tracking.

### What about copyright?
VideoGrabber is for personal, offline use only. Users must respect copyright laws and platform terms of service.

### Can I download playlists?
Yes! Check the "Download Playlist" option to download entire playlists.

### Does it support 4K?
Yes! Select "4K (2160p+)" format for ultra HD downloads.

### What if a download fails?
Check your internet connection and verify the URL is valid. Some sites may have restrictions.

### How do I update?
Download the latest version from GitHub releases and reinstall.
